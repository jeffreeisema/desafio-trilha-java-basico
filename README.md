# Desafio de Código Trilha Java Básico: Conta Banco

Este mini projeto é a resolução de um desafio proposto na Trilha Java Básico do BootCamp Santander + DIO. O desafio está disponível neste repositório [DIO - Trilha Java Básico]("https://github.com/digitalinnovationone/trilha-java-basico/tree/main/desafios/sintaxe") no GitHub.

De forma resumida, o desafio era simular a criação de uma conta Bancária simples pelo terminal, onde deveriamos exibir uma saída para o usuário requisitando os dados necessários para a abertura da conta e no final exibir uma mensagem de final.
